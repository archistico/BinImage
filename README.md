# SHADE OF COLOR
Software che converte un file in un'immagine

### UTILIZZO
#### Eseguire da go  
`go run NOMEFILE`

#### Compilazione  
`go build -o shadeofcolor.exe main.go`  
`shadeofcolor NOMEFILE`

#### Opzioni  
Per scegliere il formato desiderato dell'immagine  
`shadeofcolor NOMEFILE -f`
